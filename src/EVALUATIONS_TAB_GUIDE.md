# Evaluations Tab - View, Edit, and Print Guide

## Overview
The Evaluations tab is a dedicated space where all completed evaluations are displayed. After evaluating an applicant from the Applications tab, the evaluation will automatically appear in the Evaluations tab.

## Workflow

### 1. Evaluating an Applicant
- Go to the **Applications** tab
- Click the **"Evaluate"** button next to any application
- Fill in the LSPU evaluation form with scores
- Click **"Submit Evaluation"** when done

### 2. Viewing Evaluations
- Navigate to the **Evaluations** tab
- All evaluated applications are listed here
- Each evaluation card shows:
  - Applicant name with "Evaluated" badge
  - Position applied for
  - Email address
  - Evaluator name
  - Total score
  - Summary of all category scores (Potential, Education, Experience, Training, Eligibility, Accomplishments)

### 3. Search and Filter
- Use the search bar to find specific evaluations
- Search by:
  - Applicant name
  - Position title
  - Email address

### 4. Edit an Evaluation
- Click the **"Edit"** button (blue) on any evaluation
- The evaluation form will open with all previously entered scores pre-filled
- Make your changes
- Click **"Update Evaluation"** to save

### 5. View as PDF
- Click the **"View as PDF"** button (teal) on any evaluation
- A printable view will open showing the complete LSPU evaluation form
- Click the **"Print"** button to print or save as PDF
- The form includes:
  - LSPU logo and header
  - All evaluation categories and scores
  - Summary totals
  - Professional formatting

## Features

✅ **Centralized View**: All evaluations in one dedicated tab
✅ **Quick Search**: Filter evaluations by name, position, or email
✅ **Edit Capability**: Update evaluations anytime
✅ **PDF Export**: Professional printable format
✅ **Score Summary**: At-a-glance view of all scores
✅ **Evaluator Tracking**: See who evaluated each applicant

## Benefits

- **Separation of Concerns**: Evaluations don't clutter the Applications tab
- **Easy Access**: Find any evaluation quickly with search
- **Flexibility**: Edit evaluations as needed
- **Professional Output**: Print-ready PDF format
- **Audit Trail**: Track who evaluated each applicant
